Detailed Description

The template to creating a new Host.

There is a global list in stage.c that will have all of the availiable/supported templates. For now I commented out all but the amd64-systemd one just for testing but I will I have more avaiable after AutoGentoo is released Feel free to uncomment other ones (or add other ones)
